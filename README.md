# MoiveApp 🎬

Bu proje Flutter ile geliştirilmiş bir film uygulamasıdır.  

## Özellikler
- 🔑 Firebase Authentication ile kullanıcı girişi
- 🎥 Movie API ile film listeleri
- ⭐ Favorilere film ekleme
- 👤 Profil düzenleme
